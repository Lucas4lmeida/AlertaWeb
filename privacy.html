Política de Privacidade

Última atualização: junho de 2026

O AlertaWeb foi desenvolvido com privacidade como princípio central. A extensão não coleta dados pessoais, não rastreia o usuário e não vende nem compartilha informações com terceiros para fins comerciais. Toda análise é realizada localmente no navegador do usuário.

1. Quem somos

O AlertaWeb é uma extensão para o Google Chrome desenvolvida de forma independente, com o objetivo único de ajudar usuários a identificar sites fraudulentos, golpes de phishing e domínios suspeitos durante a navegação.

2. Propósito único da extensão

A extensão tem um único propósito: analisar o domínio da página ativa e alertar o usuário caso o site apresente características de golpe, fraude ou phishing. Isso é feito por meio de heurísticas locais, listas de domínios confiáveis e suspeitos, e consulta opcional a APIs de verificação de segurança.

3. Dados coletados

O AlertaWeb não coleta dados pessoais. As únicas informações processadas são:

URL / Domínio Analisado localmente para verificar se o site é seguro. Nunca armazenado em servidores externos.
Conteúdo da página Texto e links extraídos localmente para detectar padrões de phishing. Não é transmitido externamente, salvo para as APIs listadas abaixo (apenas o domínio/URL, nunca o conteúdo pessoal).
Preferências do usuário Configurações da extensão e histórico de análises, armazenados localmente via chrome.storage. Não saem do dispositivo.

4. Permissões utilizadas

activeTab Permite ler o URL da aba ativa no momento em que o usuário aciona a análise. Não acessa outras abas.
scripting Utilizado para injetar indicadores visuais na página atual quando um site suspeito é detectado, alertando o usuário diretamente durante a navegação.
storage Armazena localmente preferências do usuário, histórico de sites analisados e listas de domínios. Os dados ficam exclusivamente no dispositivo.
host_permissions Necessário para verificar o domínio de qualquer site visitado, comparando-o com listas de domínios confiáveis e suspeitos. Também permite o funcionamento do scanner em webmails (Gmail, Outlook, Yahoo).

5. Scanner de emails

Em clientes de webmail (Gmail, Outlook, Yahoo Mail), a extensão analisa links presentes nos emails abertos pelo usuário para identificar possíveis tentativas de phishing. Essa análise é feita inteiramente no dispositivo do usuário — nenhum conteúdo de email é transmitido para servidores externos.

6. APIs externas consultadas

Para aprimorar a precisão das análises, a extensão pode consultar as seguintes APIs. Apenas o domínio ou URL da página é enviado — nunca dados pessoais do usuário.

Google Safe Browsing API

Verifica se o domínio consta na lista de sites maliciosos do Google. Apenas a URL é enviada.

VirusTotal API

Consulta a reputação do domínio em múltiplos mecanismos antivírus. Apenas o domínio é enviado.

IP2WHOIS API

Obtém dados de registro do domínio (data de criação, registrante) para verificar domínios recentes. Apenas o domínio é enviado.

Google Gemini API

Utilizada para análise heurística avançada de conteúdo de páginas suspeitas. Fragmentos de texto da página (sem dados pessoais) podem ser enviados para análise. Nenhum código executável é recebido — a resposta é sempre texto ou JSON com score de risco.

7. O que não fazemos

Não vendemos dados Não rastreamos o usuário Não exibimos anúncios Não coletamos dados pessoais Não compartilhamos com terceiros Não transmitimos conteúdo de emails

8. Armazenamento local

O histórico de análises e as configurações da extensão são armazenados localmente no navegador do usuário via chrome.storage.local. O usuário pode apagar esses dados a qualquer momento nas configurações da extensão ou removendo-a do Chrome.

9. Menores de idade

O AlertaWeb não coleta dados de nenhuma pessoa, independentemente da idade. A extensão não é direcionada a menores de 13 anos e não solicita qualquer informação pessoal.

10. Alterações nesta política

Eventuais atualizações serão publicadas nesta página com a data de revisão atualizada. O uso continuado da extensão após a publicação de alterações implica concordância com a nova política.
